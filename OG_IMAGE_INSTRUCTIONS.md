# OG Image Creation Instructions

## Required File
Create `public/og.png` with dimensions: **1200x630 pixels**

## Design Specifications
- Background: Primary blue (#1e3a8a)
- Main text: "EduTest Global" (white, bold, large)
- Subtitle: "International Testing Center" (light blue)
- Location: "Yerevan, Armenia" (optional, smaller text)

## Quick Creation Options

### Option 1: Online Tools
1. Use [Canva](https://www.canva.com/) or [Figma](https://www.figma.com/)
2. Create a 1200x630px design
3. Export as PNG
4. Save to `public/og.png`

### Option 2: Convert SVG to PNG
1. The file `public/og.svg` is already created as a template
2. Use an online SVG to PNG converter
3. Set dimensions to 1200x630
4. Save as `public/og.png`

### Option 3: Use Image Editing Software
- Photoshop, GIMP, or similar
- Create 1200x630px canvas
- Add text and styling
- Export as PNG
