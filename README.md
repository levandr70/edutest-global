# EduTest Global — Marketing Website

A modern Next.js 15 marketing website for EduTest Global, an international testing center.

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript**
- **TailwindCSS**
- **React 19**

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [This site can’t be reached
localhost refused to connect.

### Build

Build the production version:

```bash
npm run build
```

### Start Production Server

Start the production server:

```bash
npm start
```

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - Reusable React components
- `public/` - Static assets

## Pages

- `/` - Home page
- `/testing` - Testing center overview
- `/testing/toefl` - TOEFL testing information
- `/testing/gre` - GRE testing information
- `/testing/act` - ACT testing information
- `/celta` - CELTA certification information
- `/about` - About page
- `/contact` - Contact information

## Features

- Sticky header navigation
- Responsive design (mobile-first)
- SEO metadata for all pages
- Accessible components
- Clean, professional UI
